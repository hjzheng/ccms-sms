# ccms-sms
短信编辑器


### 如何使用

```
npm install --save ccms-sms-editor
```

```js
import angular from 'angular';
import ccmsSmsEditor from 'ccms-sms-editor';

angular.module('app', [ccmsSmsEditor]);
```

```html
<!--具体用法: 参见 Demos-->
<ccms-sms-editor></ccms-sms-editor>
<ccms-sms-preview></ccms-sms-preview>
```
